# Machine Learning with Python

This section covers the transition from data manipulation to building predictive models.

## Modules

- **01 NumPy Basics**: Introduction to vectors, matrices, and efficient numerical operations.
- **02 Pandas Start**: Fundamental data analysis with Pandas (loading, cleaning, merging).
- **03 Phase 2 Project**: A complete end-to-end data analysis project (Churn Analysis).
- **04 ML Basics**: First steps with `scikit-learn` for regression and classification tasks.

## Key Libraries

- **NumPy**: The foundation for numerical computing.
- **Pandas**: High-level data manipulation tools.
- **Matplotlib / Seaborn**: Visualization libraries.
- **Scikit-Learn**: The toolkit for machine learning algorithms.

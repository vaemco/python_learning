# Data Science with Python

This section explores data manipulation and analysis using the **Pandas** library.

## Scripts

- **`pandas_basics.py`**: introduction to creating and inspecting DataFrames.
- **`data_cleaning_missing_values.py`**: handling `NaN` values and data quality issues.
- **`shop_value_calculation.py`**: performing arithmetic operations on DataFrame columns.
- **`sales_data_setup.py`**: helper script to generate sample CSV data.
- **`csv_roundtrip.py`**: saving DataFrames to CSV and reading them back.
- **`sales_analysis_report.py`**: example workflow of reading data, calculating metrics, and saving a specific report.

## Key Concepts

- **DataFrame**: The primary data structure in Pandas (like a table).
- **Data Cleaning**: Essential step before analysis (`dropna`, `fillna`).
- **IO**: Input/Output operations (CSV, Excel, etc.).
- **Vectorization**: Performing operations on entire columns efficiently without loops.

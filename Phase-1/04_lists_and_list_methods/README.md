# Lists and List Methods

This module demonstrates the usage of Python lists, including creation, manipulation, and method application.

## Contents

- **`list.py`**: shows filtering and sorting a list of numerical predictions.
- **`ml_model_queue.py`**: demonstrates using a list as a queue for model names (append and pop).
- **`todo_list_methods.py`**: covers basic list operations such as indexing, appending, popping, and iterating.

## Key Concepts

- **Lists**: Ordered, mutable collections of items.
- **Methods**: `.append()`, `.pop()`, `.remove()`, `.sort()`.
- **List Comprehension**: Concise way to create lists (shown in `list.py`).

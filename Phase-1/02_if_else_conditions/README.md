# If-Else Conditions

This module covers control flow logic in Python using conditional statements.

## Contents

- **`confidence_classifier.py`**: demonstrates a simple threshold check to simulate a confidence classifier.
- **`model_contest_selection.py`**: shows how to iterate through a list of dictionaries and select items based on a condition.
- **`score_filter_bot.py`**: illustrates filtering a list of numeric scores using a loop and conditional logic.

## Key Concepts

- **Conditionals**: `if`, `else`, `elif` statements.
- **Comparison Operators**: `>=`, `<`, `==`, etc.
- **Integration**: combining logic with loops and data structures.

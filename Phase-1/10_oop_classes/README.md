# OOP Classes

This module introduces Object-Oriented Programming (OOP) concepts in Python.

## Contents

- **`simple_model.py`**: defines a simple `Dataset` class to wrap a list of data and provide a method to get information about it.

## Key Concepts

- **Classes**: Blueprints for creating objects.
- **`__init__`**: The constructor method for initializing objects.
- **Attributes**: Variables attached to an object (`self.data`).
- **Methods**: Functions attached to an object (`get_info`).

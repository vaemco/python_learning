# Variables and Data Types

This module introduces fundamental concepts in Python programming, focusing on variables and basic data types.

## Contents

- **`intro_printing.py`**: demonstrates how to declare string variables and use f-strings for formatted output.
- **`simple_age_math.py`**: shows basic integer arithmetic using variables.

## Key Concepts

- **Variables**: Storing data values.
- **Strings**: Text data.
- **Integers**: Whole numbers.
- **f-Strings**: A way to embed expressions inside string literals for printing.

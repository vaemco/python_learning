# Dictionaries and Sets

This module introduces dictionaries, a key data structure in Python for storing key-value pairs.

## Contents

- **`ml_config_dict.py`**: demonstrates how to store and update experiment statistics.
- **`model_score_filter.py`**: shows how to filter a list of dictionaries based on specific criteria (e.g., accuracy).
- **`training_config_dict.py`**: illustrates reading, writing, and extending configuration dictionaries.
- **`vocabulary_quiz.py`**: a simple interactive quiz application using a dictionary for storage.

## Key Concepts

- **Dictionaries**: Collection of key-value pairs (`{"key": "value"}`).
- **Key-Value Access**: Retrieving values by key.
- **Iterating**: Looping through keys, values, or items.

# Pandas Start

This module dives deeper into **Pandas** for data manipulation and analysis in a machine learning context.

## Scripts

- **`data_inspection.py`**: basic inspection methods (`head`, `info`, `describe`).
- **`data_loading.py`**: loading data and simple filtering.
- **`dataframe_basics.py`**: manual DataFrame creation.
- **`cleaning_data.py`**: handling missing values.
- **`filtering_drill.py`**: exercising data selection with complex conditions.
- **`grouping_stats.py`**: using `groupby` for aggregations.
- **`merging_data.py`**: joining multiple DataFrames.
- **`plotting_basics.py`**: basic plotting with Pandas and Matplotlib.
- **`seaborn_heatmap.py`**: visual correlation analysis.
- **`sales_pipeline.py`**: a more complex, class-based example of a data processing pipeline.

## Key Concepts

- **Filtering**: Selecting rows based on conditions (`&`, `|`).
- **Grouping**: Aggregating data (`groupby().mean()`).
- **Merging**: Combining datasets (`pd.merge`).
- **Visualization**: Plotting data directly from Pandas or using libraries like Seaborn.

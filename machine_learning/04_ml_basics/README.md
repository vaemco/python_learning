# Machine Learning Basics

This module introduces fundamental **Machine Learning** concepts using `scikit-learn`.

## Scripts

- **`first_prediction.py`**: A simple Linear Regression (predicting continuous values).
- **`diabetes_check.py`**: Logistic Regression for binary classification.
- **`credit_bot.py`**: Decision Tree Classifier.
- **`train_test_split_demo.py`**: The importance of splitting data into training and testing sets.
- **`visualize_regression.py`**: Plotting the regression line against data points.
- **`warmup_recap.py`**: A quick refresher on the training API.

## Key Concepts

- **Scikit-Learn**: The industry-standard library for classical ML in Python.
- **Model API**: `model.fit(X, y)` to train, `model.predict(X)` to predict.
- **Classification vs. Regression**: Predicting categories vs. numbers.
- **Evaluation**: Testing the model on unseen data.

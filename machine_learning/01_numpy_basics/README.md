# NumPy Basics

This module introduces **NumPy**, the fundamental package for scientific computing in Python.

## Scripts

- **`vector_basics.py`**: working with 1D arrays (vectors), element-wise operations.
- **`dot_product.py`**: calculating the scalar product of two vectors (e.g., scoring models).
- **`matrix_ops.py`**: working with 2D arrays (matrices) and matrix-vector multiplication.

## Key Concepts

- **Arrays**: Efficient storage for numerical data.
- **Vectorization**: Performing operations on whole arrays at once.
- **Dot Product**: A key linear algebra operation (`np.dot`).

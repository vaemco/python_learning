# Functions and Parameters

This module explains how to define and use functions in Python to create reusable code blocks.

## Contents

- **`loss_functions.py`**: implements a simple loss calculation function.
- **`random_scores_simulation.py`**: demonstrates generating random data and processing it with a function.
- **`scope_demo.py`**: explores the concept of variable scope (local vs. global variables).
- **`scoring_function.py`**: shows how to separate logic into a function for better readability and reusability.

## Key Concepts

- **Functions**: `def` keyword.
- **Parameters**: Inputs to functions.
- **Return Values**: Outputs from functions.
- **Scope**: Where variables are visible and accessible.

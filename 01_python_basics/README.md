# Phase 1: Python Basics

This section of the repository covers the fundamental concepts of Python programming. Each subfolder corresponds to a key topic, progressing from variables to object-oriented programming.

## Modules

- **01 Variables and Data Types**: Strings, Integers, f-strings.
- **02 If/Else Conditions**: Logic, Thresholds, Comparisons.
- **03 Loops (For and While)**: Iteration, Input loops.
- **04 Lists and List Methods**: Storage, Sorting, Filtering.
- **05 Dictionaries and Sets**: Key-Value pairs, Lookups.
- **06 Functions and Parameters**: Reusable logic, Scope, Return values.
- **07 List Comprehensions**: Concise functional-style list creation.
- **08 Error Handling (Try/Except)**: Robustness, Input validation.
- **09 File Handling**: Reading from and writing to files.
- **10 OOP Classes**: Basics of Object-Oriented Programming (Classes, Objects, Methods).
- **11 Final Project**: A mini-pipeline combining the learned concepts.
